# Skill: Análisis de Escenarios

## Objetivo
Crear y comparar múltiples escenarios financieros para evaluar el impacto de diferentes decisiones y condiciones de mercado.

## Escenarios Predefinidos

### Optimista (+20% ventas)
- Ventas aumentan 20% sobre la proyección base
- Costos variables suben proporcionalmente
- Costos fijos se mantienen
- Representa: Campaña exitosa, nuevo cliente grande, temporada excepcional

### Pesimista (-30% ventas)
- Ventas caen 30% bajo la proyección base
- Costos variables bajan proporcionalmente
- Costos fijos se mantienen (rigidez)
- Representa: Pérdida de cliente clave, crisis económica, competencia agresiva

### Inflacionario (+15% costos)
- Costos fijos y variables suben 15%
- Ventas se mantienen (no se puede traspasar al precio)
- Representa: Inflación de insumos, devaluación, aumento de arriendos

### Crecimiento Acelerado (+50% ventas, +30% costos)
- Ventas crecen 50% (nuevos clientes)
- Costos fijos suben 30% (más personal, local más grande)
- Representa: Expansión, nueva sucursal, contratación

## Métricas de Comparación
Para cada escenario calcular:
- Caja mínima y mes en que ocurre
- Necesidad de financiamiento
- Meses con caja negativa
- Break-even
- Runway

## Formato de Salida
Tabla comparativa con escenario base vs alternativas, destacando diferencias críticas.
